  Samba - 01 (Windows用ファイルサーバー)
  
[内容]
  ## 作業内容
    ホストをWindows用ファイルサーバーに構築する
  
  ## 使用マシン
    HOST : localhost        # ホスト名を「FS01」に変更して識別する
    IPV4 : 192.168.1.3
    MEMO : ホスト(CentOS7)  # Windows用ファイルサーバーに構築する
    USER : 追加なし         # OSユーザーではなくSambaユーザー登録が必要
  
  ## 使用マシン
    HOST : localhost
    IPV4 : 192.168.1.2
    MEMO : 家庭内LANのノート(Windows10)  # ドメイン未参加(WORKGROUP)
    USER : Jill, Kate, Mike, John        # 追加しておく
  
  ## 設定情報(Samba)
    DOMAIN : WORKGROUP -> Windows側が「WORKGROUP」なので必然
    DIRECTORY : /
    SHARE-NAME : share
