  File System - 1 (設定ファイルの書式)
  
[内容]
  ## 設定ファイル(/etc/fstab)の書式
  --------------------------------------------------
  デバイス名(UUID名)/ラベル名  マウントポイント  ファイルシステム  マウントオプション  dump対象  fsck順序
  
  (設定例)
  /dev/sda1  /boot  ext2  defaults  0  2
  
  ファイルシステム
    EXT2      : EXTの拡張版
    EXT3      : ジャーナリング機能付加
    EXT4      : 機能拡張
    ReiserFS  : Linux用ジャーナリングファイルシステム
    JFS       : IBM製ジャーナリングファイルシステム
    XFS       : SGI製ジャーナリングファイルシステム -> CentOS7のデフォルト
    btrfs     : 先進的ファイルシステム
    ISO9660   : CD-ROM用
    UDF       : DVD用
    F2FS      : SSD用
    MSDOS     : MS-DOS用
    FAT/FAT32 : フラッシュメモリ用(USB)
    NTFS      : WindowsNT/2000以降
    HPFS      : OS/2用
    HFS       : MacOS用
    HFS+      : MacOS8.1以降
    NFS       : ファイル共有(Linux)
    CIFS      : ファイル共有(Windows)
    procfs    : プロセス情報を扱う仮想ファイルシステム(/proc)
    sysfs     : デバイス情報を扱う仮想ファイルシステム
    tmpfs     : 仮想メモリを扱う仮想ファイルシステム
    devpts    : 疑似端末制御のための仮想ファイルシステム
    usbfs     : USBデバイス監視用
    cramfs    : 組込システム用
  
  マウントオプション
    async     : 入出力同期
    sync      : 入出力同期
    atime     : Iノードアクセス時刻更新
    relatime  : Iノードアクセス時刻更新
    noatime   : Iノードアクセス時刻更新  
    auto      : コマンド「mount -a」の実行時の動作
    noauto    : コマンド「mount -a」の実行時の動作
    exec      : バイナリ実行
    noexec    : バイナリ実行
    suid      : SUID, SGID
    nosuid    : SUID, SGID
    ro        : 読み取り専用
    rw        : 読み取り専用
    user      : 一般ユーザーのマウント
    users     : 一般ユーザーのマウント
    nouser    : 一般ユーザーのマウント
		--------------------------------------------------
		その他
			owner				// デバイスファイル所有者のマウント許可
			defaults		// async, auto, dev, exec, nouser, rw, suid
			dev					// ファイルシステム上のデバイスファイル使用可
			group				// 条件「ユーザーのグループ = デバイスファイルの所有グループ」が満たされた場合にマウント許可
			uid=UID		// 全ファイル所有者を指定UIDにする
			gid=GID		// 全ファイル所有グループを指定GIDにする
	
	dump対象
		0						// コマンド「dump」のバックアップ対象外
		0以外				// コマンド「dump」でバックアップ
	
	fsck順序
		0						// システム起動時のコマンド「fsck」のチェック対象外
		0以外				// システム起動時のコマンド「fsck」のチェック順序(1, 2‥の順)
  --------------------------------------------------
