  DNSクライアントコマンド - 03 (dig)
  
[内容]
  ## 作業内容
    コマンド(dig)でDNSサーバーに名前解決を依頼する
  
  ## 使用マシン
    HOST : vm12
    IPV4 : 192.168.122.12
    MEMO : 仮想マシン
  
  ## コマンド(dig)
    dig -x      // 逆引き名前解決を行う
    dig -p XX   // 問合先ポート(XX)を指定
    dig +norec  // 再帰問合せをしない
    dig +rec    // 再帰問合せをする
    dig +noall  // 全セクション非表示
    dig +all    // 全セクション表示
    dig +noquestion  // 質問セクション非表示
    dig +question    // 質問セクション表示
    dig +noanswer    // 回答セクション非表示
    dig +answer      // 回答セクション表示
