[内容]
  ## 作業内容
    ソースの修正部分(パッチ)の反映をコマンド(patch)で行う
  
  ## 使用マシン
    HOST : vm12
    IPV4 : 192.168.122.12
    MEMO : 仮想マシン(サーバー用途)
  
  ## コマンド(patch)
    patch -C  // 処理内容のテスト
    patch -R  // 処理内容を取り消して元に戻す
    patch -p0 XX < YY  // ファイル(XX)にパッチ(YY=patch/patch01/AAA)の差分を適用、パスを修正しない   -> patch/patch01/AAA
    patch -p1 XX < YY  // ファイル(XX)にパッチ(YY=patch/patch01/AAA)の差分を適用、最初の'/'まで削除  -> patch01/AAA
    patch -p2 XX < YY  // ファイル(XX)にパッチ(YY=patch/patch01/AAA)の差分を適用、二番の'/'まで削除  -> AAA
